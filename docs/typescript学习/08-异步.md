# 08 异步与类型

目标：为异步代码正确添加类型，处理 Promise 与 fetch 的类型。

Promise 类型：

```ts
function fetchJson(url: string): Promise<any> { /* ... */ }
```

使用 `async/await`：

```ts
async function getUser(id:number): Promise<User> {
  const r = await fetch(`/api/${id}`);
  return r.json();
}
```

为 `fetch` 返回值建模：

```ts
type ApiResponse<T> = { code: number; data: T }
async function call<T>(url:string): Promise<T> { /* parse ApiResponse */ }
```
