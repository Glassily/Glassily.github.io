# 03 函数与类型签名

目标：函数类型、可选/默认参数、重载与 `this` 类型。

函数声明与类型：

```ts
function add(a: number, b: number): number { return a + b; }
```

可选与默认参数：

```ts
function greet(name: string, title?: string) {}
function pow(x: number, n = 2) {}
```

函数类型与接口：

```ts
type Fn = (a: number, b: number) => number;
const f: Fn = (x,y) => x+y;
```

重载：

```ts
overload function parse(x: string): string[];
overload function parse(x: number): number;
function parse(x: any) { /* impl */ }
```

注意 `this` 类型：在回调/事件中明确 `this` 可以避免错误。
