# TypeScript 学习指南

欢迎！本教程面向有 JavaScript 基础，想系统学习 TypeScript 的工程师。学习目标：理解类型系统、掌握常见模式、能在项目中安全使用 TypeScript。

章节目录：

- [01-入门](01-入门.md) — 为什么使用 TS、如何快速配置
- [02-类型系统](02-类型.md) — 原始类型、联合、交叉、字面量
- [03-函数与类型签名](03-函数.md) — 函数类型、重载、this、泛型函数
- [04-接口与类型别名](04-接口.md) — `interface` vs `type`、扩展与实现
- [05-类与面向对象](05-类.md) — 类、修饰符、抽象类、继承
- [06-泛型进阶](06-泛型.md) — 约束、默认类型、条件类型示例
- [07-模块与打包](07-模块.md) — ES 模块、命名/默认导出、声明文件
- [08-异步与类型](08-异步.md) — Promise、async/await、fetch 类型
- [09-进阶类型技巧](09-进阶.md) — 映射类型、工具类型、类型推断
- [10-练习与示例](10-练习.md) — 实战练习与参考答案
- [resources](resources.md) — 推荐资料

学习建议：每章阅读并在本地运行示例，完成练习后对照参考答案。
