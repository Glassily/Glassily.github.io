# 07 模块与打包

目标：理解模块系统、导出/导入与声明文件（.d.ts）。

导出示例：

```ts
// foo.ts
export function sum(a:number,b:number){return a+b}
export default class A {}
```

导入示例：

```ts
import A, { sum } from './foo'
```

声明文件：当使用 JS 库时，若无类型声明，可编写 `declarations.d.ts`：

```ts
declare module 'some-old-lib' {
  export function old(x:any): any;
}
```
