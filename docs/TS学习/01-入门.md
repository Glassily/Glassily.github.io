# 01 入门：为什么选择 TypeScript

目标：理解 TypeScript 带来的好处并完成本地配置。

- TypeScript 是 JavaScript 的超集，增加静态类型检查。
- 改善大型项目的可维护性、IDE 自动补全、重构安全性。

快速开始（使用 npm）：

```bash
npm init -y
npm install -D typescript
npx tsc --init
```

在 `tsconfig.json` 中建议开启：

```json
{
  "compilerOptions": {
    "target": "ES2019",
    "module": "ESNext",
    "strict": true,
    "esModuleInterop": true,
    "skipLibCheck": true,
    "forceConsistentCasingInFileNames": true
  }
}
```

将 `.js` 文件改为 `.ts` 后编译：

```bash
npx tsc
```
