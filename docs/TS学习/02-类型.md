# 02 类型系统基础

目标：掌握基本类型、联合、交叉与类型推断。

基础类型：

```ts
let a: number = 1;
let b: string = 'hello';
let c: boolean = true;
let d: any = 123; // 避免使用 any
```

数组与元组：

```ts
let nums: number[] = [1,2,3];
let pair: [string, number] = ['age', 30];
```

联合类型与字面量类型：

```ts
function id(x: number | string) { return x; }
let direction: 'up' | 'down' = 'up';
```

类型别名与交叉类型：

```ts
type A = {x: number};
type B = {y: string};
type C = A & B; // {x: number, y: string}
```

类型推断：TypeScript 会在很多情况下推断类型，推荐开启 `strict`。
